# Шаги для запуска
1. git clone <этого репозитория>
2. Установить и запустить postgresql
3. Отредактировать config.ini файл в соответствии с настройками postgresql (user, password)
4. Выполнить скрипт sql/init.sql
5. Установить python 3.12+
6. Выполнить pip install -r requirements.txt
7. Настроить в "планировщике задач" задачи, в соответствии со скриншотами из папки img
